# Neon Slider

This is an AI-powered air hockey game built with Next.js and rendered on a Canvas. It features a sleek neon aesthetic, particle effects, and multiple difficulty levels for the computer opponent.

## Features

- **Vibrant Neon Visuals**: A retro-inspired look with glowing paddles, a trailing puck, and colorful rink borders.
- **Dynamic Physics**: A responsive game engine for fast-paced action and crisp collisions.
- **Player vs. Computer (PVC)**: Challenge an AI opponent with four different difficulty levels:
    - Easy
    - Medium
    - Hard
    - Expert
- **Player vs. Player (PVP)**: Go head-to-head against a friend on the same device.
- **Built with Next.js**: Modern, performant, and ready to be deployed.

## Getting Started

To get started, clone the repository and install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
