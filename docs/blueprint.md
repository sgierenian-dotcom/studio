# **App Name**: Neon Slider

## Core Features:

- Paddle Movement: Enable the players to move the paddles to hit the puck.
- Puck Physics: Model puck physics to determine how the puck bounces off the walls and paddles.
- Score Tracking: Track the scores of the two players.
- AI Opponent: Implement an AI tool opponent that the player can play against, with an adjustable difficulty level.

## Style Guidelines:

- Primary color: Neon green (#39FF14) to emulate the neon aesthetic of the original prompt.
- Background color: Dark gray (#212121) to make the neon elements stand out.
- Accent color: Purple (#a73aff) for secondary interactive elements.
- Body and headline font: 'Space Grotesk' (sans-serif) for a futuristic, sporty look, paired with 'Inter' for readability if blocks of text longer than a line or two are anticipated.
- Use simple, geometric icons for buttons and interactive elements.
- Maintain a clean, centered layout with ample spacing around the game elements.
- Subtle glow animations to the paddles and puck.
